🚀 COMMENT TESTER :

2. Cliquer sur le bouton de verrouillage (🔒) en haut à droite
3. Entrer le mot de passe (1234 par défaut)
4. Aller dans l'onglet "Paramètres"
5. Utiliser le toggle "Police cursive pour les titres"
6. Retourner aux écrans étudiants pour voir l'effet
7. Recharger la page pour vérifier la persistance

📋 CODES COULEURS DU TOGGLE :
- Gris (#ccc) = Désactivé (police normale)
- Bleu (#3498db) = Activé (police cursive)

🎨 POLICES CURSIVES UTILISÉES :
- Brush Script MT (Windows)
- Lucida Handwriting (Windows) 
- Segoe Print (Windows)
- cursive (fallback universel)

💾 SAUVEGARDE :
- Clé localStorage : 'cursiveFontEnabled'
- Valeur : 'true' ou 'false' (string)
